# 26.05 Release notes (unreleased)

```{default-domain} nix
```

## Breaking changes

No breaking change were introduced in this EPNix release.

## New features and highlights

- {pkg}`epnix.channel-finder-service` was upgraded to `5.0.0+`
  and {pkg}`epnix.support.reccaster` / {pkg}`python3Packages.recceiver` were upgraded to `1.9.2+`.
  No user-visible breaking changes.

- {pkg}`epnix.phoebus` now bundles the Phoebus documentation,
  and follows a file structure closer to what's distributed in official Phoebus packages.

## Fixes

- spring-boot based services
  (ChannelFinder, Phoebus Olog, Phoebus save-and-restore)
  now won't start an Embedded LDAP server
  unless the Embedded LDAP is explicitly used.
  This solves a port conflict when installing those services
  on the same machine.

  This fix was backported to the `nixos-25.11` branch.

## Documentation

- The {doc}`../ioc/user-guides/testing/python-scripts` article was rewritten
  and is now much simpler.
